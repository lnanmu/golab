## 编程的基础概念

* 什么是程序：完成某个功能的指令的集合。我来指挥我的电脑来干活，但是电脑听不懂人类的语言，所以我写了一段代码，让电脑去执行，可是电脑只认识0和1，不认识其他的语言，所以我要吧代码进行编译，得到的是可执行文件，电脑执行可执行文件就完成了任务，在这个过程中，代码就是程序。
